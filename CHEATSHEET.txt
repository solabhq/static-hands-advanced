NAVIGATION (CapsLock +)
────────────────────────────────────────────────────────────
  I/K/J/L  = ↑/↓/←/→ Arrows
  U / O    = Home / End
  Y / N    = Page Up / Down
  H        = Delete
  ;        = Backspace
  P        = Insert

TEXT EDITING (CapsLock +)
────────────────────────────────────────────────────────────
  C = Copy        Z = Undo
  V = Paste       A = Select All
  X = Cut         B = Bold

MODIFIERS - Hold (CapsLock +)
────────────────────────────────────────────────────────────
  W = Win         S = Alt
  D = Shift       F = Ctrl
  
  Example: CapsLock + D + L = Shift + Right (select)

SPEED NAVIGATION ⚡ (CapsLock +)
────────────────────────────────────────────────────────────
  8 = Up 5 lines       , = Down 5 lines
  M = Left 6 words     . = Right 6 words

CASE TRANSFORM (Select text first, then CapsLock +)
────────────────────────────────────────────────────────────
  9 = UPPERCASE        0 = lowercase        - = Title Case

SPECIAL (CapsLock +)
────────────────────────────────────────────────────────────
  Q = Escape      7 = Tab         ` = Close app
  R = Rename      4 = $$          = = Always on top
  T = Select line
